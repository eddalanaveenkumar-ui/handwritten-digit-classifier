# Data Directory

Place your MNIST dataset files here:

- `train.csv` - Training data with labels
- `test.csv` - Test data (optional)

## Dataset Format

The CSV files should have the following format:
- First column: `label` (0-9)
- Remaining columns: `pixel0` to `pixel783` (28x28 = 784 pixels)

## Download MNIST Dataset

You can download the MNIST dataset from:
- Kaggle: https://www.kaggle.com/c/digit-recognizer
- Official source: http://yann.lecun.com/exdb/mnist/
